**제목**: 취향 경매
**유형**: 시뮬레이션
**내용**:
- 목적: 내가 뭘 진짜 좋아하는지 경쟁 상황에서 드러내기
- 소요시간: 25분 / 인원: 20~30명 (모둠 상관없이 개인전)
- 준비물: 없음 (가상 점수 1,000점, 칠판에 낙찰 기록)
- 진행방법:
  1. 각자 1,000점을 가짐
  2. 강사가 항목을 하나씩 경매 — 평생 무제한으로 주는 조건. 예: 게임/유튜브 무한 시청권, 잠 푹 자기, 친구랑 노는 시간, 맛집 프리패스, 내 방 꾸미기, 외모 업그레이드, 운동 능력, 돈, 인기, 자유 시간
  3. 원하는 항목에 점수 입찰 → 최고가 낙찰. 쓴 점수는 사라짐
  4. 전략적으로 배분해야 하니 "다 갖고 싶은데 못 갖는" 상황이 핵심
- 디브리핑 질문:
  1. 제일 많이 베팅한 게 뭐였어? 왜 거기에 몰빵했어?
  2. 놓쳐서 아까웠던 건? 그게 의외로 나한테 중요한 거였을 수도
  3. 점수 다 쓰고 보니, 나는 뭘 우선으로 두는 사람인 것 같아?
- 주의사항: 낙찰 못 받아도 "입찰 안 한 것도 선택"이라고 짚어주기
**출처**: 가치관 경매(Values Auction) 기법을 청소년 관심사로 변형
**태그**: 취향탐색, 우선순위, 의사결정

**제목**: 입장 정하기 (한 줄 서기)
**유형**: 신체활동
**내용**:
- 목적: 몸을 움직여 내 성향을 눈으로 확인
- 소요시간: 20분 / 인원: 20~30명 (전체)
- 준비물: 없음 (교실 양 끝 = 척도 양극)
- 진행방법:
  1. 한쪽 벽 "완전 그렇다", 반대쪽 "전혀 아니다"
  2. 문장 제시하면 자기 생각 위치로 이동
  3. 문장 예: "계획파보다 즉흥파다" / "단톡방 알림 바로 확인한다" / "혼자 노는 게 편하다" / "벼락치기 vs 미리 준비" / "여행은 빡빡한 일정 vs 발길 닿는 대로"
  4. 같은 줄에 선 사람끼리 30초 이유 공유 → 몇 명 발표
- 디브리핑 질문:
  1. 어디서 사람들이랑 제일 갈렸어? 그게 너의 어떤 면인 것 같아?
  2. 위치 정하기 애매했던 문장 있었어? 왜 망설였어?
  3. 친구 이유 듣고 "어 저것도 말 되네" 했던 거 있어?
- 주의사항: 정답 없음 강조, 소수 쪽에 선 사람 존중 분위기
**출처**: Spectrum/Barometer 토론기법 변형
**태그**: 성향탐색, 자기인식, 신체활동

**제목**: 공통점 빙고
**유형**: 창의놀이
**내용**:
- 목적: 나의 사소한 특징을 꺼내고, 의외의 공통점 발견
- 소요시간: 25분 / 인원: 20~30명
- 준비물: 없음 (각자 종이에 3×3 직접 그림, 없으면 빈 종이 한 장이면 됨)
- 진행방법:
  1. 9칸에 자기 특징·경험 하나씩 적기. 예: "라면은 꼬들" "최애 가수 있음" "왼손잡이" "아침형 인간" "분식 vs 치킨파" "겜BTI 있음"
  2. 돌아다니며 같은 칸을 가진 친구 찾아 사인 받기 — 한 칸당 다른 사람
  3. 가로/세로/대각선 한 줄 완성하면 빙고
- 디브리핑 질문:
  1. 의외로 나랑 같은 친구 누구였어?
  2. 아무도 사인 못 해준 칸 있어? 그건 너만의 특이 포인트일 수도
  3. 9칸 채우고 보니 나는 어떤 거에 진심인 사람 같아?
- 주의사항: 가족·상처·성적 같은 민감 항목은 피하고 가벼운 취향 위주로 안내
**출처**: Human Bingo / Find Someone Who 변형
**태그**: 공통점찾기, 관계형성, 취향탐색
